  •  package Kalah

    The Kalah framework for the AI course at FAU WS16/17.

    The Kalah framework for the AI course at FAU WS16/17. This package contains:

    • The abstract info.kwarc.teaching.AI.Kalah.Agents.Agent class, which is supposed to be extended by your agents.
    • A info.kwarc.teaching.AI.Kalah.Agents.HumanPlayer agent you can use to test your agents
    • A info.kwarc.teaching.AI.Kalah.Board class representing a game board
    • A info.kwarc.teaching.AI.Kalah.Game class that lets two agents play each other on a new board.

Agent 

abstract class Agent extends AnyRef

The abstract class for agents. Needs to implement four things:

- Agent.name : What you call your agent (just for fun). Please make this one fixed (unlike in the example below) - Agent.students : Please put your full names in this list. - Agent.init : Called at the start of a game. - Agent.move : is called when it's the agent's turn
